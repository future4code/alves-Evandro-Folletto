PROJETO BLACKJACK:

Descrição:

Blackjack ou Vinte-e-um é um jogo em que o objetivo é ter mais pontos do que o adversário, mas sem ultrapassar os 21. Neste projeto, o usuário e o computador recebem duas cartas cada. Apenas uma carta do computador é revelada, e então o jogador escolhe se irá solicitar outra carta ou não. Vence aquele que obter mais pontos, sem ultrapassar 21. A iteração do usuário é feita através do navegador e a lógica toda feita utilizando JavaScript.

Tecnologias Utilizadas:

Para a realização deste projeto, foram utilizadas as tecnologias HTML5 e JavaScript.
