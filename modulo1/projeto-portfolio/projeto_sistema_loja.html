PROJETO SISTEMA LOJA:

Descrição:

Foi criada uma página de uma loja de materiais de construção. O usuário pode navegar entre as páginas: a) página inicial - composta apenas por um texto de apresentação. b) controle de estoque - foi adicionada uma tabela contendo alguns produtos comercializados pela loja. c) adicionar produto - foi colocado um formulário para que o usuário possa iteragir. d) funcionários - adicionada uma lista com os nomes dos funcionarios separados por setor. e) vendas - foi adicionado um formulário para o usuário iteragir e a partir desta mesma aba é possível acessar o histórico de vendas.

Tecnologias Utilizadas:

Para a realização deste projeto, foram utilizadas as tecnologias HTML5 e CSS.
