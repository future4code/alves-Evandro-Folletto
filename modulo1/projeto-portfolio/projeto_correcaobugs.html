PROJETO CORREÇÃO BUGS:

Descrição:

Este projeto tem por finalidade a correção de alguns bugs que são reportados pelo problema. É necessário que sejam feitas as correções (chamadas de "issues") e dados um commit para cada uma, relatando brevemente a solução adotada. O exercício é composto por onze exercícios e dois desafios. Para a realização desta tarefa se faz necessário uma atenção especial no arquivo CSS, pois lá estão as configurações da página.

Tecnologias Utilizadas:

Para a realização deste projeto, foram utilizadas as tecnologias HTML5 e CSS.
