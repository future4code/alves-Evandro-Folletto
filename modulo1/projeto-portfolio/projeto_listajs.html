PROJETO LISTA JAVASCRIPT:

Descrição:

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. Este projeto consiste em uma lista de exercício relacionado ao assunto de JavaScript. A lista consiste em dois exemplos de testes, 12 exercícios e 3 desafios. Os assuntos abordados são funções, operadores lógicos, arrays, objetos, entre outros. Para a resolução desta lista, existe uma iteração com o usuário através do prompt.

Tecnologias Utilizadas:

Para a realização deste projeto, foram utilizadas as tecnologia HTML e JavaScript.
